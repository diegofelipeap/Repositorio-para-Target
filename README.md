## Questão 1: Sequência de Fibonacci

### Explicação

Este programa verifica se um número fornecido pertence à sequência de Fibonacci. A sequência de Fibonacci começa com 0 e 1, e cada número subsequente é a soma dos dois anteriores (0, 1, 1, 2, 3, 5, 8, ...). O código utiliza um loop para calcular os números da sequência até o número informado ou até que o número seja encontrado na sequência.

### Como Executar

1. Salve o código em um arquivo chamado `fibonacci.js`.
2. Abra o arquivo no Visual Studio Code.
3. Use o atalho de teclado `Ctrl + Alt + N` (Windows/Linux) ou `Cmd + Option + N` (Mac) para executar o código com a extensão **Code Runner**.
4. O programa irá exibir se o número fornecido pertence ou não à sequência de Fibonacci no painel de saída.

3. Use o atalho de teclado `Ctrl + Alt + N` (Windows/Linux) ou `Cmd + Option + N` (Mac) para executar o código com a extensão **Code Runner**.

## Questão 5: Identificação de Interruptores e Lâmpadas

### Explicação

Este programa simula um cenário em que três interruptores controlam três lâmpadas em salas diferentes. A lógica utiliza o estado de "quente" e "frio" das lâmpadas para determinar qual interruptor controla qual lâmpada, usando apenas duas visitas às salas das lâmpadas.

### Como Executar

1. Salve o código em um arquivo chamado `lampadasInterruptores.js`.
2. Abra o arquivo no Visual Studio Code.
3. Use o atalho de teclado `Ctrl + Alt + N` (Windows/Linux) ou `Cmd + Option + N` (Mac) para executar o código com a extensão **Code Runner**.
4. O programa irá simular o processo descrito e exibirá o resultado de cada etapa, identificando qual interruptor controla qual lâmpada no painel de saída.
